O que é o VRaptor3?

VRaptor 3 é um framework MVC para web
focado no desenvolvimento ágil.

Através da inversão de controle e injeção
de depêndencias ele corta drasticamente o
tempo de trabalho que seria perdido com o
código repetitivo: validações, conversões,
direcionamentos, ajax e lookups.
